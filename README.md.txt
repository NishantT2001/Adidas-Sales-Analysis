# Adidas Sales Analysis 📊

This project analyzes Adidas sales data across various regions, retailers, and sales methods using an Excel dataset.

## 📂 Dataset Overview
The Excel file contains:
- Raw sales data (transactions, regions, sales method, profit, etc.)
- Aggregated views: sales trends, sales by region, method, and retailer

## 🧠 Analysis Insights
- Top retailers: Amazon, Walmart
- Sales trends by month for 2020
- Sales performance by region
- Sales method comparison: In-store vs Online

## 📁 Files
- `Adidas-Sales-Analysis.xlsx` — Main dataset

## 📌 Tools
- Excel
- (Optional) Python (Pandas, Matplotlib/Seaborn)
